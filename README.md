# Almoxarifado SENAC

**Deploy:** https://universidade-cesumar.github.io/prova-2bi-ads-3sem-guilhermegouve4/

## Sobre o projeto

Sistema de controle de estoque desenvolvido para o laboratório de enfermagem do SENAC. O projeto nasceu de um levantamento de requisitos feito com a instrutora responsável pelo almoxarifado, que hoje faz o controle manualmente em planilhas do Excel.

## Funcionalidades

- Cadastro de novos materiais com nome e quantidade
- Retirada e abastecimento de estoque diretamente na linha de cada item
- Busca de materiais pelo nome
- Total de itens cadastrados atualizado em tempo real
- Sinalização visual de itens com estoque crítico (menos de 10 unidades)
- Todas as operações salvas via API REST (MockAPI)

## Tecnologias

- HTML, CSS e JavaScript puro.

## Como rodar

Abra o arquivo `index.html` no navegador ou acesse o link de deploy acima.
